# Command reference

## Very basic system details

| command      | Purpose                                  |
|--------------|------------------------------------------|
| whoami       | User that is logged in                   | 
| hostname     | Server's name                            |
| pwd          | Where you are located in the system      |
| logout       | Exit the session                         |
| exit         | Exit the session                         |
| CRTL+D       | Exit the session                         |
